# E2E テスト

## 自動 E2E

Playwright ベースの自動 E2E を追加しています。

### 実行方法

```bash
npm run e2e
```

### 現在の対象

- demo モード起動
- AI 分類候補の生成
- 分類パターン切り替え
- プレビュー上のドラッグ編集
- 保存リクエストへの編集結果反映

### 補足

- 自動 E2E は `client` の demo モードを使い、API は Playwright 側でモックします。
- 実ブラウザの強制終了や実ファイル書き換えは行いません。

## 手動確認の推奨シナリオ

1. `client` と `server` を起動する
2. 各ブラウザのブックマークを読み込めることを確認する
3. ルート選択を切り替え、AI 整理プレビューが生成されることを確認する
4. プレビュー上で並び替えができることを確認する
5. 保存開始後にブラウザ再起動が走り、再表示後に保存ステータスが確認できることを確認する
6. 未選択ルートが保持されていることを確認する
7. ロールバックで直前バックアップへ戻せることを確認する
