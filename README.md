# Browser Bookmarkbar Synchronizer (ブラウザ・ブックマークバー・同期ツール)

PCにインストールされている主要なブラウザ間で、ブックマークバーの内容をシームレスに同期・整理するためのブラウザベース・ツールです。

## 概要

本プロジェクトは、複数のブラウザ（Google Chrome, Microsoft Edge, Braveなど）を使い分けるユーザーが、それぞれのブックマークバーを一元管理し、最適な状態に同期することを目的としています。直感的なドラッグ＆ドロップ操作に加え、AIによるラベル要約や自動整理機能を備えています。

## 主な機能

### 1. マルチブラウザ対応
* **対象ブラウザ:** Chrome, Edge, Braveなどの主要なブラウザをサポート。
* **バーティカル表示:** 各ブラウザのブックマークバーの内容を、ブラウザごとに縦に並べて一覧表示します。

### 2. 直感的な編集機能
* **ドラッグ＆ドロップ:** ブックマークを掴んで別のブラウザの列や、同一ブラウザ内の別の場所に自由に移動できます。
* **重複チェック:** 同一のURLを持つブックマークは自動的に識別され、整理をサポートします。

### 3. スマート・同期ツール
* **「マージ」ボタン:** すべてのブラウザに共通する（最大公約数的な）ブックマークセットをAIとアルゴリズムで自動生成し、統一されたブックマークバーを作成します。
* **「要約」ボタン (AI連携):** ブックマークのラベル名が長すぎる場合、AIがその内容を分析し、一目でわかる短い単語に自動で要約します。
* **「整理」ボタン:** すべてのブックマークをフォルダごとに、名前順（昇順）で自動ソートします。

### 4. ブラウザへの反映
* **「保存」ボタン:** ツール上で編集した最終的な内容を、ボタン一つですべてのローカルブラウザのブックマークバーへ書き戻します。

## 技術スタック

* **フロントエンド:** React
* **スタイリング:** CSS (Vanilla CSS / モダンデザイン)
* **ロジック:** JavaScript / Node.js (ブラウザ用APIとの調整)
* **AI要約:** Gemini API (予定)

## 開発の動機

複数のブラウザを併用していると、特定のブラウザだけで保存したブックマークが他のブラウザに反映されず、作業効率が低下することがあります。本ツールは、そのような「ブックマークの散らばり」を解消し、常に最適なブラウジング環境を維持するために開発されました。
